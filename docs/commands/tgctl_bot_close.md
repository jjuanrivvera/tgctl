## tgctl bot close

Close the bot instance before moving it to another server

### Synopsis

Close the bot instance (frees server resources). Returns an error for the first 10 minutes after the bot launches.

```
tgctl bot close [flags]
```

### Examples

```
  tgctl bot close
```

### Options

```
  -h, --help   help for close
```
