## tgctl alias set

Create or update an alias

```
tgctl alias set <name> <expansion> [flags]
```

### Examples

```
  tgctl alias set ping "bot info"
  tgctl alias set notify "message send --chat @me"
```

### Options

```
  -h, --help   help for set
```
