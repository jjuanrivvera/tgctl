## tgctl doctor

Diagnose configuration, credentials, and connectivity

### Synopsis

Run a series of checks (config, token, API reachability, clock) and exit non-zero if any fail.

```
tgctl doctor [flags]
```

### Examples

```
  tgctl doctor
  tgctl doctor --json
```

### Options

```
  -h, --help   help for doctor
      --json   output checks as JSON
```
